# moreutils

My reimplementation of some of the fantastic utilities from `moreutils`

## Implemented

- [x] `chronic`
- [x] `combine`
- [x] `errno`
- [x] `ifne`
- [ ] `isutf8`
- [ ] `mispipe`
- [ ] `parallel`
- [x] `pee`
- [x] `sponge`
- [x] `ts`
- [x] `vidir`
- [x] `vipe`
- [ ] `zrun`

## Will not implement

* `lckdo` - use `flock`
* `ifdata` - use `ip` or `ifconfig`
